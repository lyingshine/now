# NOW - 职业成长平台

一个帮助用户实现职业目标的学习管理系统。

## 🚀 Vue 重构中

项目正在从原生 HTML/CSS/JS 重构为 Vue 3 + Vite。

### 当前功能
- 💰 实时收入计算器
- 💼 职位任务系统（21个职位）
- 🌱 成长中心（学习计划和进度追踪）
- 📊 生活水平对比
- 🎯 任务奖励系统

### 技术栈（重构后）
- Vue 3
- Vite
- Pinia (状态管理)
- Vue Router
- TailwindCSS

## 开发

```bash
# 安装依赖
npm install

# 开发服务器
npm run dev

# 构建
npm run build
```

## 🎯 任务奖励系统优化

### 每个职位包含6项技能
每项技能都有：
1. **技能描述** - 详细的能力要求
2. **💰 任务奖励** - 完成该技能可获得的薪资增长
3. **🎯 成长任务** - 具体、可量化的学习目标

### 示例：前端工程师（¥18,000）

| 技能 | 奖励 | 任务 |
|------|------|------|
| HTML/CSS/JS 基础 | +3,000元 | 30个布局练习 + FreeCodeCamp认证 |
| Vue3/React18 | +4,000元 | 3个完整项目（电商、后台、H5） |
| 前端工程化 | +2,500元 | 搭建脚手架 + 优化打包 + CI/CD |
| TypeScript | +2,000元 | 重构2个项目 + 高级类型 |
| 性能优化 | +2,500元 | Lighthouse 90+ 优化 |
| 协作规范 | +2,000元 | 开源贡献 + 测试覆盖率80% |

**总计：18,000元** ✅

## 🚀 使用方法

### 1. 访问职位页面
```
http://127.0.0.1:3000/jobs.html
```

### 2. 浏览职位列表
- 14个职位可选
- 显示薪资、段位、标签
- 已接取任务有绿色徽章

### 3. 点击职位卡片
- 查看详细要求
- 查看生活水平对比 ⭐
- 查看成长任务
- 勾选完成的技能

### 4. 接取任务
- 完成所有6项技能要求
- 点击"接取任务"按钮
- 数据同步到 NOW 收入计算器

## 📊 进度计算逻辑

### 技能进度
```
技能进度 = 已完成任务数 / 总任务数 × 100%

示例：
技能1有 12 个任务，完成了 6 个
进度 = 6 / 12 × 100% = 50%
```

### 整体进度
```
整体进度 = 所有技能进度的平均值

示例：
技能1: 100%
技能2: 50%
技能3: 0%
技能4: 0%
技能5: 0%
技能6: 0%

整体进度 = (100 + 50 + 0 + 0 + 0 + 0) / 6 = 25%
```

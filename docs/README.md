## 🚀 快速导航

### 新手入门
1. 阅读 [项目 README](../README.md) 了解项目概况
2. 查看 [系统架构文档](./ARCHITECTURE.md) 理解整体架构
3. 参考 [样式架构文档](./STYLE_ARCHITECTURE.md) 学习样式系统

### 开发指南
- **添加新功能** → 参考系统架构文档的模块架构部分
- **修改样式** → 参考样式架构文档的最佳实践部分
- **性能优化** → 查看系统架构文档的性能优化章节
- **组件开发** → 参考组件架构和样式组件库章节

### 常见问题

#### Q: 如何添加新的设计令牌？
A: 在 `src/styles/tokens/design-tokens.css` 中添加新的 CSS 变量，参考样式架构文档的设计令牌系统章节。

#### Q: 如何创建新的组件？
A: 参考系统架构文档的组件架构章节，遵循命名规范和代码组织原则。

#### Q: 如何添加新的页面路由？
A: 在 `src/router/index.js` 中添加路由配置，参考系统架构文档的路由架构章节。

#### Q: 如何管理应用状态？
A: 使用 Pinia Store，参考系统架构文档的状态管理章节。

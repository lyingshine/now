## 🚀 如何使用

### 1. 接取任务

首先，在**任务大厅**（jobs.html）接取一个任务：

1. 访问 http://127.0.0.1:3000/jobs.html
2. 浏览 21 个职位
3. 点击任意职位查看详情
4. 勾选完成所有 6 项技能要求
5. 点击"接取任务"按钮

✅ **系统会自动生成详细的学习计划！**

---

### 2. 查看成长中心

接取任务后，访问**成长中心**：

1. 点击导航栏的"🌱 成长中心"
2. 或直接访问 http://127.0.0.1:3000/growth-center.html

你会看到：

## 📊 学习计划生成逻辑

### 自动生成规则

接取任务后，系统会自动：

1. **为每项技能生成 4 周学习计划**
   - 第 1 周：基础学习
   - 第 2 周：实践项目
   - 第 3 周：进阶学习
   - 第 4 周：综合应用

2. **解析成长任务**
   - 从职位的 `growthTask` 字段提取任务
   - 按 `+` 号分割成具体步骤
   - 分配到 4 周学习计划中

3. **计算完成时间**
   - 默认每周学习 10 小时
   - 6 项技能 × 4 周 = 24 周（约 6 个月）

### 示例：前端工程师

**技能 1：HTML/CSS/JS 基础** (+3,000元)
```
第1周：HTML/CSS/JS 基础 - 基础学习
- 完成 30 个页面布局练习
- 通过 FreeCodeCamp 前端认证
- 手写常见 JS 原生方法

第2周：HTML/CSS/JS 基础 - 实践项目
- 继续实践...

第3周：HTML/CSS/JS 基础 - 进阶学习
- 进阶内容...

第4周：HTML/CSS/JS 基础 - 综合应用
- 综合项目...
```

## 🔧 技术实现

### 文件结构
```
growth-center.html          # 成长中心页面
growth-center-data.js       # 数据模型和学习计划生成器
growth-center-app.js        # 应用逻辑
jobs-app.js                 # 更新：接取任务时生成学习计划
index.html                  # 更新：添加成长中心导航
jobs.html                   # 更新：添加成长中心导航
```

### 核心函数

#### 生成学习计划
```javascript
growthCenterData.generateLearningPlan(job)
```

#### 获取学习计划
```javascript
growthCenterData.getPlan(jobId)
```

#### 更新进度
```javascript
growthCenterData.updateProgress(planId)
```
